# Full-Stack Login Application

This is a full-stack **Login and Signup** application built to understand end-to-end web development.

## Project Overview

- **Frontend:** Developed using **Angular**, providing a responsive and interactive user interface.
- **Backend:** Built with **C# and .NET**, handling authentication logic and API integration.
- **Database:** User data is securely stored and managed using **SQL**, completing the full-stack architecture.

## Features

- User registration (Signup)
- User login (Authentication)
- Secure password storage
- API-driven architecture


## Installation

### Frontend

```bash
cd auth-demo-ui
npm install
ng serve
```

Frontend runs on http://localhost:4200

### Backend

```bash
cd AuthDemoAPI
dotnet restore
dotnet run
```

Backend runs on http://localhost:5051 (or your configured port)
